## Calabash Panel
Calabash panel is an UI dashboard for [calabash](https://github.com/approach0/calabash).

Let us orchestrate! :bowtie:

:guitar: :musical_keyboard: :violin: :drum:

### Local hosting
```
$ npm install
$ npm run watch
```

### Production
```
# CALABASH_URL=http://localhost:8964
# npm run build
# npm run serve
```
